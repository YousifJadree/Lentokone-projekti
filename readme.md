Lentokonepeli (Yousif, Khalid, Zejnel)
Pelin idea: 

Pelaaja matkustaa Euroopan ympäri, isojen lentokenttien läpi. 
Peli antaa 3 vihjettä maasta joka arvauksen jälkeen. Ensimmäinen vihje väärän arvauksen jälkeen: Lentokentän nimi. Toinen vihje: Maan väkiluku. Kolmas vihje: Lipun värit.
Pisteytys: Oikea arvaus ensimmäisellä yrityksellä: 3 pistettä. Oikea arvaus toisella yrityksellä: 2 pistettä jne.
Terminaali näyttää jäljellä olevat maat ja poistaa ne sen mukaan kun ne on käytetty.
Pelaaja voi lopettaa pelin milloin vain kirjoittamalla "quit" tai olla jatkamatta seuraavalle lentokentälle "(y/n)".
Peli ei vielä tallenna aiempia pisteitä, eikä käytä nimike kirjautumista.
